## ⏰ PROTOCOLO 9:00 AM — 10:30 AM (90 minutos total)

### 9:00 — Desinfecta SAB (10 min)

```
1. Abre SAB
2. Limpia interior con papel + alcohol 70%
   → Pasa por paredes, tapa, piso interior
3. Cierra tapa
4. Espera 10 minutos (se seque, se evapore alcohol)
   → Mientras esperas, revisa que todo esté listo sobre mesa
```

### 9:10 — Inoculación bolsa #1 (15 min para aprender)

```
ABRE SAB lentamente (no brusco, menos aire entra)

1. Coloca bolsa #1 vacía DENTRO SAB
2. Abre boca bolsa (cuidado, movimientos lentos)
3. Mete cucharada:
   → 30g sustrato (base de bolsa)
   → 15g spawn Rosa (distribuído)
4. MEZCLA DENTRO BOLSA:
   → 30 segundos, mueve bien
   → Objetivo: spawn uniforme, no grumos
5. SELLA BOLSA:
   → Cierra boca fuerte
   → Nudo triple O sellador térmico
   → LENTO, sin aire adentro
6. ETIQUETA INMEDIATO:
   "Rosa_Ent1_Filtro_Jun18 | Bolsa #1"
7. SACA bolsa del SAB
8. FOTO: bolsa sellada (baseline)

TIEMPO TOTAL BOLSA #1: ~15 min (es lenta, aprendes)
```

### 9:25 — Inoculación bolsas #2–5 (10 min por bolsa, más rápido)

```
REPITE BOLSA #1 para #2, #3, #4, #5
MÁS RÁPIDO cada vez (sin ir a lo loco, aséptico)

Bolsa #2: 9:25–9:35 (10 min)
```

## ⚠️ SI ALGO FALLA DURANTE INOCULACIÓN

| Problema | Solución |
|---|---|
| **Bolsa se abre accidentalmente en SAB** | Desecha bolsa, abre nueva, continúa. Es dato: "Esto pasó, riesgo de contaminación aquí" |
| **Spawn se derrama** | Limpia SAB, continúa. Usa menos spawn en bolsa siguiente si necesario. |
| **No selladora térmica funciona** | Usa nudo triple fuerte. Válido, solo más lento. |
| **SAB huele a ácido/mal después desinfectar** | Normal (alcohol). Espera más tiempo que seque. |
| **No tienes papel periódico** | Usa periódico/toalla, no es crítico. |

**Si algo GRAVE**: Pausa, respira, documenta qué pasó, continúa mañana.
