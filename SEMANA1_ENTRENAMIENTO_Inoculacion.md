## MARTES 18 JUNIO — ENTRENAMIENTO 1: Rosa Batch DIY

### 08:00 — Setup Inoculación

**Antes de tocar SAB, prepara fuera**:

```
MATERIALES LISTOS (sobre mesa):
✓ 3 bolsas PP 20×50 vacías (o 4 si quieres espacio error)
✓ Sustrato: 150g total
  → OPCIÓN A: Papa cocida desmenuzada + agua (mock-sustrato, realista)
  → OPCIÓN B: Aserrín cocido + agua (más auténtico)
  → OPCIÓN C: Sustrato real si lo tienes pre-preparado
✓ Spawn Rosa: 100g total (~33g por bolsa × 3)
✓ Cucharas/espátulas limpias (para mezclar)
✓ Marcador + papel (etiquetas)
✓ Papel periódico (desorden)
```

### 09:00 — Protocolo SAB (Primera inoculación real)

```
PASO 1: Prepara SAB
- Limpia interior con papel + alcohol 70%
- Cierra tapa
- Espera 10 minutos (se seque)
- Abre lentamente
- ✓ Verifica: ¿Olor a químico? (confirmación hermético)

PASO 2: Inoculación (dentro SAB, cuidado asepsia)
1. Coloca 3 bolsas vacías en interior SAB
2. Abre boca de bolsa #1 (cuidado con el aire)
3. Añade ~50g sustrato (capas base)
4. Añade ~33g spawn Rosa (distribuído)
5. Mezcla dentro bolsa (30 segundos, movimientos rápidos)
6. SELLA bolsa:
   - Opción A: Nudo triple fuerte (clásico)
   - Opción B: Sellador térmico si tienes
   - Opción C: Cinta (menos ideal, pero válido)
7. Etiqueta INMEDIATAMENTE: 
   "Rosa_Entrenamiento1_Jun18 | 33g spawn + sustrato"
8. Repite pasos 2–7 para bolsa #2 y #3
9. Saca bolsas del SAB

OBSERVACIÓN durante proceso:
¿Qué salió bien?
- ¿SAB hermético sin problemas?
- ¿Brazos caben sin rozar paredes?
- ¿Pudiste mezclar sin abrir bolsa?
¿Qué fue difícil?
- ¿Fue lenta la mezcla? (riesgo contaminación)
- ¿Se abrió bolsa accidentalmente?
- ¿Sellaste rápido o fue lento?
```

```
RECOMENDACIÓN: A (Tenjo) → datos más útiles futuros
```
